# Assignment 1 - Proposition de projet ML

## Mon projet :
Prédire le résultat d'un match de football (victoire, nul, défaite)

## Le business case :
Aider les parieurs ou les clubs sportifs à anticiper les résultats 
des matchs pour optimiser leurs décisions stratégiques.

## Les sources de données :
- Kaggle : https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017

## Type de problème ML :
Classification (3 classes : victoire / nul / défaite)

## Objectif d'évaluation :
Maximiser la précision (accuracy) et le F1-score sur les prédictions
